# Mahji — Interactive Prototype

Interactive wireframe prototype for the Mahji mahjong app.

## Pages
- **Home** — stats, navigation cards, decorative branding
- **Learn** — filterable lessons (First Timer → Advanced)
- **Practice** — filterable drills with card selector
- **Play** — difficulty, card selector, mat color picker, game board with N/S/E/W racks
- **Shop** — 5 categories with drill-in product pages
- **Ask Bam Bird** — AI chat mentor (full page + popup overlay on Learn/Practice)
- **Profile** — stats, edit profile with goals, preferences, sign in/out

## Local Development
```
npm install
npm run dev
```

## Deploy
Push to GitHub, then connect to **Netlify** or **Vercel** — both auto-detect Vite and deploy instantly.

## Design System
- **Colors:** Cherry (#E03050), Lavender Deep (#7E64A4), Seafoam (#6DBFA8), Cerulean (#8EC7E2)
- **Fonts:** Bodoni Moda (titles), Outfit (body)
- **Mat Colors:** Coffee, Seafoam, Lavender, Cerulean
